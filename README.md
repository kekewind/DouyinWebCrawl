# Douyin Web Crawl
**1分钟抓取抖音网页版用户资料**
- 在抖音网页版上爬数据
## 工具
- python
- selenium

## 步骤：
1. 设置好chromedriver。下载与chrome相匹配的chromedriver。[Download Link](https://chromedriver.storage.googleapis.com/index.html)

2. 运行douyin_crawl.ipynb，弹出chrom，手动滑块验证并登入账号。


3. 输入用户关键词，比如“关键词”


4. 接着运行douyin_crawl.ipynb并保存*.csv文件中
